Knowledge Base (Documents)
Raw documents that the system can retrieve information from.

Document Chunking
Documents are split into smaller, manageable chunks for better retrieval.
⬇
3
Vector Embedding
Chunks are converted into high-dimensional vectors using an embedding model.
⬇
4
Query Embedding
Your question is also converted into a vector in the same embedding space.
→
⬇
5
Similarity Search
The system finds chunks whose vectors are most similar to the query vector.
⬇
6
Context Retrieval
Top-k most relevant chunks are retrieved to provide context for the answer.
⬇
7
Prompt Construction
Retrieved context is injected into a prompt template along with the original question.
⬇
8
LLM Generation
The LLM uses the augmented prompt to generate a grounded, accurate answer.
